# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using transactional data from **3,900 purchases** across multiple product categories.
The objective is to identify **customer segments, revenue drivers, and product trends** to support data-driven business decisions.

The project combines **Python (EDA), SQL (business analysis), and Power BI (dashboard visualization).**

---

## Dataset Summary

* **Total Transactions:** 3,900
* **Total Columns:** 18
* **Average Purchase Amount:** $59.76
* **Average Customer Rating:** 3.75

### Key Data Features

* Customer demographics (Age, Gender, Location, Subscription Status)
* Purchase details (Item Purchased, Category, Season, Size, Color)
* Shopping behavior (Discounts, Frequency, Shipping Type, Ratings)

## Data Cleaning & Preparation

Data preprocessing was performed using Python:

* Standardized column names to **snake_case**
* Handled **37 missing review ratings** using category median
* Created new features:

  * `age_group`
  * `purchase_frequency`
* Removed redundant column `promo_code_used`

---

## SQL Business Analysis

SQL queries were used to answer key business questions such as:

* Revenue generated by **gender**
* Customers using **discounts but spending above average**
* **Top 5 products** by rating
* **Shipping type comparison**
* **Subscription vs non-subscription spending**
* **Top products per category**
* **Customer segmentation (New / Returning / Loyal)**

---

## Key Insights

* Male customers generated **$157,890 revenue**, more than double female revenue.
* **Clothing category** produced the highest revenue and sales.
* **Young adults** contributed the highest revenue among age groups.
* Only **27% of customers are subscribers**, indicating a strong growth opportunity.
